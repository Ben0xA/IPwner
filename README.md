# IPwner
Get the owner of IP Addresses from ARIN. It attempts to see if it was reassigned to a third-party.

![IPwner Usage](/ipwner.png?raw=true "IPwner Usage")

## Requirements
```
apt install whois
```

## Make it executable
```
chmod +x ipwner.sh
```

## Usage
```
./ipwner.sh <ip>
./ipwner.sh <filename>
```
